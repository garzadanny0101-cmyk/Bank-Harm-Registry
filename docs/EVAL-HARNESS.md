# 95% Eval Harness

Must pass before public traffic:
- Homepage loads on mobile
- `/api/health` returns JSON
- Fake report submits
- Email or GitHub issue is received
- Consent is required
- Sensitive SSN/card/account-like data is blocked
- Turnstile is active and server-validated
- No public story posts automatically
- Privacy, terms, disclaimer, donation disclaimers are linked
- Donation page says support is not tax-deductible unless qualified
- Volunteer expert page does not promise testimony or case outcomes
