# Legal Remedies Map

Educational map only, not legal advice.

Common issue → possible route:
- Credit reporting error → CFPB, credit bureau/furnisher dispute, FTC if identity theft.
- Debt not owed → CFPB, FTC/state AG depending collector.
- Checking/savings problem → CFPB and bank regulator.
- Account freeze/closure → CFPB, OCC/FDIC/Fed/state regulator depending charter.
- Unauthorized transfer/scam → CFPB, FTC, bank fraud department, law enforcement where appropriate.
- Investment/brokerage → SEC/FINRA.
- Credit union → NCUA.

Remedy options:
- refund/reimbursement
- correction
- written explanation
- account access
- stop collections/reporting
- preserve records
- investigation results
